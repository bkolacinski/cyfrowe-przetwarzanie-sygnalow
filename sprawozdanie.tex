\section{Cel zadania}

Celem zadania jest implementacja aplikacji do generacji i analizy sygnałów
oraz szumów cyfrowych. Aplikacja umożliwia generowanie jedenastu różnych
rodzajów sygnałów, wykonywanie podstawowych operacji arytmetycznych
między sygnałami (dodawanie, odejmowanie, mnożenie, dzielenie),
obliczanie parametrów statystycznych oraz prezentację histogramu
rozkładu wartości. Dodatkowo zaimplementowano obsługę zapisu
i odczytu sygnału do/z pliku binarnego.

\section{Wstęp teoretyczny}

Sygnał dyskretny jest sekwencją wartości uzyskaną przez
próbkowanie sygnału ciągłego z częstotliwością
próbkowania $f_s$~\cite{oppenheim1999discrete}. Zgodnie z twierdzeniem
Nyquista--Shannona, aby możliwa była wierna rekonstrukcja sygnału,
częstotliwość próbkowania musi być co najmniej dwukrotnie wyższa
od najwyższej częstotliwości składowej sygnału.

Do podstawowych parametrów statystycznych sygnału należą
\cite{proakis2006dsp}:

\begin{itemize}
  \item Wartość średnia:
    \begin{equation}
      \bar{y} = \frac{1}{N} \sum_{n=0}^{N-1} y[n]
      \label{eq:mean}
    \end{equation}

  \item Wartość średnia bezwzględna:
    \begin{equation}
      \overline{|y|} = \frac{1}{N} \sum_{n=0}^{N-1} |y[n]|
      \label{eq:mean_abs}
    \end{equation}

  \item Wartość skuteczna (RMS -- Root Mean Square):
    \begin{equation}
      y_{\mathrm{RMS}} = \sqrt{\frac{1}{N} \sum_{n=0}^{N-1} y[n]^2}
      \label{eq:rms}
    \end{equation}

  \item Wariancja:
    \begin{equation}
      \sigma^2 = \frac{1}{N} \sum_{n=0}^{N-1} \bigl(y[n] - \bar{y}\bigr)^2
      \label{eq:variance}
    \end{equation}

  \item Moc średnia:
    \begin{equation}
      P = \frac{1}{N} \sum_{n=0}^{N-1} y[n]^2
      \label{eq:power}
    \end{equation}
\end{itemize}

Dla sygnałów okresowych obliczenia statystyczne są wyznaczane
na podstawie pełnych okresów, co pozwala na uniknięcie błędów
wynikających z niekompletnych cyklów~\cite{lyons2011dsp}.

\subsection{Eksperyment nr 1 -- Generacja sygnałów podstawowych}

\subsubsection{Założenia}

Zaimplementowano następujące rodzaje sygnałów:

\begin{itemize}
  \item \textbf{S1} -- Szum o rozkładzie jednostajnym
  \item \textbf{S2} -- Szum gaussowski
  \item \textbf{S3} -- Sygnał sinusoidalny
  \item \textbf{S4} -- Sygnał sinusoidalny wyprostowany jednopołówkowo
  \item \textbf{S5} -- Sygnał sinusoidalny wyprostowany dwupołówkowo
  \item \textbf{S6} -- Sygnał prostokątny
  \item \textbf{S7} -- Sygnał prostokątny symetryczny
  \item \textbf{S8} -- Sygnał trójkątny
  \item \textbf{S9} -- Skok jednostkowy
  \item \textbf{S10} -- Impuls jednostkowy
  \item \textbf{S11} -- Szum impulsowy
\end{itemize}

\subsubsection{Przebieg}

Sygnał sinusoidalny (S3) opisany jest równaniem:
\begin{equation}
  y(t) = A \cdot \sin\!\left(\frac{2\pi}{T}(t - t_1)\right)
  \label{eq:sine}
\end{equation}
gdzie $A$ jest amplitudą, $T$ okresem podstawowym, a $t_1$ czasem
początkowym.

Sygnał sinusoidalny wyprostowany jednopołówkowo (S4) przyjmuje
postać:
\begin{equation}
  y(t) = \begin{cases} A \cdot \sin\!\left(\frac{2\pi}{T}(t - t_1)\right) &
  \text{jeśli } \sin(\cdot) > 0 \\ 0 & \text{w przeciwnym razie}
  \end{cases}
  \label{eq:sine_half}
\end{equation}

Sygnał prostokątny (S6) z współczynnikiem wypełnienia $k_w$:
\begin{equation}
  y(t) = \begin{cases} A & \text{jeśli } (t \bmod T) < k_w \cdot T \\
  0 & \text{w przeciwnym razie} \end{cases}
  \label{eq:square}
\end{equation}

Szum gaussowski (S2) generowany jest z rozkładu normalnego
$\mathcal{N}(0,\, A/3)$.

\subsection{Eksperyment nr 2 -- Parametry statystyczne sygnału}

\subsubsection{Założenia}

Dla każdego wygenerowanego sygnału wyznaczane są następujące
parametry według wzorów (\ref{eq:mean})--(\ref{eq:power}): wartość
średnia, wartość średnia bezwzględna, wartość skuteczna RMS,
wariancja oraz moc średnia. Dla sygnałów okresowych obliczenia
wykonywane są na dokładnie pełnych okresach.

\subsubsection{Przebieg}

Implementacja funkcji liczącej parametry statystyczne:

\begin{lstlisting}
def calculate_signal_metrics(signal):
    y = np.asarray(signal)
    power = np.mean(np.square(y))
    return {
        "mean":     np.mean(y),
        "mean_abs": np.mean(np.abs(y)),
        "rms":      np.sqrt(power),
        "variance": np.var(y),
        "power":    power,
    }
\end{lstlisting}

Dla sygnałów okresowych wykrywana jest liczba pełnych okresów
w wybranym przedziale czasu, a obliczenia ograniczane do tych danych.

\subsection{Eksperyment nr 3 -- Operacje na sygnałach}

\subsubsection{Założenia}

Aplikacja umożliwia składanie dwóch lub więcej sygnałów
za pomocą czterech operacji arytmetycznych: dodawania, odejmowania,
mnożenia i dzielenia. Każda operacja wykonywana jest próbka po próbce
na wektorach o równej długości.

\subsubsection{Przebieg}

Operacje realizowane są następującymi funkcjami:

\begin{lstlisting}
def signal_add(y1, y2):
    min_len = min(len(y1), len(y2))
    return y1[:min_len] + y2[:min_len]

def signal_multiply(y1, y2):
    min_len = min(len(y1), len(y2))
    return y1[:min_len] * y2[:min_len]

def signal_divide(y1, y2):
    min_len = min(len(y1), len(y2))
    # replace zeros with small value to avoid division by zero
    y2_safe = np.where(y2[:min_len] == 0, 1e-10, y2[:min_len])
    return y1[:min_len] / y2_safe
\end{lstlisting}

Dzielenie zabezpieczone jest przed dzieleniem przez zero poprzez
zastąpienie zerowych wartości małą wartością $10^{-10}$.

\subsubsection{Rezultat}

Mnożenie sygnału sinusoidalnego przez sygnał prostokątny
pozwala uzyskać modulację amplitudy (AM). Dodawanie dwóch
sygnałów sinusoidalnych o bliskich częstotliwościach
prowadzi do efektu bicia.

\subsection{Eksperyment nr 4 -- Zapis i odczyt sygnału}

\subsubsection{Założenia}

Aplikacja umożliwia zapis wygenerowanego lub przetworzonego sygnału
do pliku binarnego (format msgpack) oraz jego późniejsze wczytanie.
Plik zawiera wektor czasu, wektor wartości oraz metadane.

\subsubsection{Przebieg}

Serializacja sygnału odbywa się przy użyciu biblioteki
\texttt{msgpack}. Struktura zapisanego pliku:

\begin{lstlisting}
data = {
    "t":        t.tolist(),
    "y":        y.tolist(),
    "metadata": metadata,
    "version":  "1.0",
    "format":   "msgpack",
}
\end{lstlisting}

\subsubsection{Rezultat}

Zapis i odczyt działa poprawnie dla wszystkich 11 typów
sygnałów. Weryfikowana jest spójność długości wektorów
$t$ i $y$ oraz niepustość danych.

\section{Wnioski}

Zrealizowana aplikacja pozwala na generowanie i analizę jedenastu
różnych typów sygnałów dyskretnych. Wyniki eksperymentów
potwierdzają poprawność implementacji wzorów matematycznych:
wartości RMS i mocy średniej sygnału sinusoidalnego są zgodne
z wartościami teoretycznymi ($y_{\mathrm{RMS}} = A/\sqrt{2}$,
$P = A^2/2$). Zastosowanie pełnych okresów do obliczeń parametrów
sygnałów okresowych eliminuje błędy związane z końcowym
efektem okna. Mechanizm zabezpieczenia przed dzieleniem przez zero zapewnia
stabilność numeryczną operacji arytmetycznych.
